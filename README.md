# LLM Agent Bug Study

All datasets used in this project are stored in the [DB](https://github.com/AnonymousRepo12/LLM-Agent-Bug-Study/tree/main/DB) folder.

## Installation

1. Download and install **Redis** on your local machine.
2. Install the project dependencies:
_pip install -r requirements.txt_

## Running the Agent

1. Start the Redis server:
_redis-server_
2. Run the agent:
_python run_agent.py_


## Notes

- Make sure Redis is running before starting the agent.
- All required Python packages are listed in `requirements.txt`.

The analysis of the dataset is present in the [anonymous site](https://sites.google.com/view/bugsinllmagents)
